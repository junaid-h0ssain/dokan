# Dokan e-commerce api
Trying to build an e-commerce api using java spring-boot

Using maven for dependencies, so using intellij idea dependecies can be auto-added

### To run this
- Clone the project
- Run using `Shift+F10` (Intellij idea)
- If not then open the terminal into the project folder and run `./mvnw spring-boot:run`

### Requirements
- Java 25 or higher (if not then change maven version in `pom.xml`)

### How to validate
Use [Postman](https://www.postman.com/downloads/) locally,     

### Database 

The Dokan API supports three database systems: H2 (in-memory), PostgreSQL, and MySQL. Use H2 for general testing. Additional apps need to be set up for others.
